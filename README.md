# Banking Power Pivot Analysis Project

## Project Overview

This project analyzes a banking dataset containing 500,000 customer, account, transaction, loan, and branch records using Microsoft Excel Power Pivot and Pivot Tables.

The objective was to answer 31 business-focused analytical questions commonly encountered in banking and financial analytics.

---

## Dataset Size

- 500,000+ Records
- Multiple Related Tables
- Customer Data
- Account Data
- Transaction Data
- Loan Data
- Branch Data

---

## Data Model

Relationships were established between:

Customers → Accounts

Accounts → Transactions

Branches → Accounts

Customers → Loans

using Excel Power Pivot Data Model.

## Business Questions Solved

The project answers 31 business questions including:

1. Which customer segment holds the highest total account balance?
2. Which city contributes the highest total deposits?
3. Which region has the largest number of customers?
4. What is the average account balance by account type?
5. Which branches manage the highest total deposits?
6. Which customer age group contributes the most deposits?
7. What percentage of customers belong to each segment?
8. Which account type is most popular?
9. Which city has the highest average customer balance?
10. What is the customer-to-account ratio by segment?
11. Which transaction channel is used most frequently?
12. What is the total transaction amount by channel?
13. Which branch generates the highest transaction volume?
14. What is the average transaction amount by channel?
15. Which region generates the highest transaction value?
16. What percentage of transactions are credit vs debit?
17. Which customer has the highest transaction activity?
18. Which loan type has the highest total loan amount?
19. What percentage of loans are active vs closed?
20. What is the average customer value by branch?
21. Which regions are underperforming?
22. Identify the top 1% customers.
23. Which customer segment is most profitable?
24. Which customers should be targeted for premium banking services?
25. Which region should receive expansion investment?
26–31. Additional banking performance and business intelligence analyses.
